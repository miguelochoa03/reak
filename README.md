## Description:
*Can’t spell freakware without reak.*

We aim to combine various features from PEAK (Aggro Crab and Landfall Games) with R.E.P.O. (SemiWork Studios). 
PEAK is a multiplayer survival game where you and your friends are stranded on an island and need to traverse
multiple biomes to reach the peak and be rescued. R.E.P.O. is a multiplayer horror game where you explore
various modern-buildings turned ruins in search of high valuable human relics in order to amass enough resale
value to continue to more difficult levels.
